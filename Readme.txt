Prerequisites :
1) Install Visual studio 2019 or Later
2) Open VS Go to Extension -> Click on Manage Extension --> Install Specflow 


Step 1 - Clone the project using git clone -b master https://github.com/suruchi10/DOTNetCORE.git

step 2 - Open project go to Accolite.DotNetCore and launch Accolite.DotNetCore.sln

step 3 - Go to Solution Explorer --> Expand Automation.API.Tests --> Expand Dependencies --> Go to Project
         Right click and add project reference
         Add Automation.API.Accelerators and Automation.Reporting project referrence

Step 4 - Clean the project

step 5 - Build the project
